## How to use

0. Write comments as described in `doc-gen.py` and make sure that your poject is structured as follows:
	1. ```
		a_folder/
			function.rell
            module.rell
            operation.rell
            query.rell
        main.rell
        chromia.yml
        etc..
	   ```
2. Clone or copy `doc-gen.py` into the `root/` of your project.
3. Run `python3 doc-gen.py`
